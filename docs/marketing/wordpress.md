## Post content (Gutenberg-friendly blocks as plain text)

### Introduction

Self-hosting an assistant like **OpenClaw** with **Telegram**, **Discord**, or other chat surfaces means managing **tokens**, **pairing**, **allowlists**, and **gateway uptime**. Doing that on the same Mac you use for email and photos is a security trade many of us refuse.

### What we published

The **OpenClaw ↔ SNS Playbook** is a **Next.js** application you can run locally or deploy. It provides:

1. **100 skills** across **Preplan**, **Plan**, **Execution**, **Post-plan**, and **Other** (including macOS file-sharing bridges when Universal Clipboard is not available across Apple IDs).
2. **Channel selectors** for Telegram, Discord, Slack, Matrix, Signal, WhatsApp, iMessage, and a local web UI concept.
3. **Ten prompts** per selection: channel × lifecycle stage × audience (**Basic**, **Intermediate**, **Advanced**), each with a **Copy** button.

### Who it is for

- Solo builders running a **24/7 MacBook Pro** as an agent host  
- People who want **structured prompts** before touching production chats  
- Readers who already use official docs but want a **single navigation map**

### Get started

Clone or download the project, run `npm install` and `npm run dev`, then open the local URL. Full quickstart: see the repository **README** and **docs/QUICKSTART.md**.
