# 티스토리 — 포스트 예시 (Korean)

## 제목

[개발/인프라] 전용 Mac에서 OpenClaw·텔레그램·디스코드 돌릴 때 쓰는 100 스킬 플레이북

## 요약 한 줄

에이전트용 맥과 일상용 맥을 나누고, 채널별로 복사해 쓰는 프롬프트 10개씩 모아둔 Next.js 웹앱 소개.

## 본문

### 들어가며

OpenClaw 같은 **자가 호스팅 에이전트**를 **텔레그램/디스코드**에 붙이면 BotFather 토큰, 디스코드 인텐트, 페어링 승인 같은 이슈가 한꺼번에 옵니다. 문서를 읽어도 “우리 집 맥 환경”에 맞는 체크리스트는 따로 필요했습니다.

### 플레이북이 하는 일

- 왼쪽: **100개 스킬**을 단계(Preplan ~ Other)와 소분류로 탐색
- 위쪽: **채널**(텔레그램, 디스코드 등) + **프롬프트 단계** + **난이도** 선택 → **프롬프트 10개** 표시 + **복사** 버튼

### 왜 웹앱인가

로컬에서 `npm run dev`로 띄우거나, 빌드해서 호스팅하면 됩니다. 데이터베이스 없이 정적에 가깝게 동작합니다.

### 맺음말

토큰은 **절대** 블로그 댓글이나 스크린샷에 넣지 마세요. 전용 맥 계정·FileVault·게이트웨이 로그 습관부터 잡는 게 먼저입니다.
