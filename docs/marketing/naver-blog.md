# 네이버 블로그 — 예시 글 (Korean)

## 제목

**전용 맥북에서 OpenClaw + 텔레그램/디스코드 쓰는 분들을 위한 100가지 스킬 플레이북**

## 본문

셀프호스팅 AI 어시스턴트(**OpenClaw** 등)를 **텔레그램·디스코드**에 연결할 때, **일상용 맥**과 **에이전트 전용 맥**을 분리하지 않으면 토큰·페어링 코드가 섞이면서 사고 위험이 커집니다.

이번에 공개한 **웹 플레이북(Next.js)** 은 다음을 한 화면에 모았습니다.

- **100개 스킬**: 사전 계획(Preplan) → 계획(Plan) → 실행(Execution) → 사후 관리(Post-plan) → 기타(맥 간 SMB, Tailscale 등)
- **채널별 프롬프트**: 텔레그램, 디스코드, 슬랙, 매트릭스 등
- **난이도**: 입문(Basic) / 중급(Intermediate) / 고급(Advanced)마다 **복사 가능한 프롬프트 10개**

**맥끼리 Apple ID가 다를 때** iCloud·유니버설 클립보드로는 연결이 안 되는 경우가 많습니다. 그럴 때 **SMB 공유 폴더**나 **Tailscale**로 프롬프트만 안전하게 옮기는 흐름도 “Other” 단계에 넣었습니다.
